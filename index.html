Neural Network Inference on Pure Silicon

FPGA-Based Digit Recognition – 97% Accuracy in 2.2ms

Built a 4-layer MLP (784→128→64→32→10) implemented as pure RTL running on an Altera DE2 Cyclone II FPGA. The FPGA receives hand-written digits via USB-UART, unpacks INT4-quantized weights from ROM, runs 111,000 MAC operations through ReLU layers, and displays the result on the HEX seven-segment display — all in 2.2ms. Achieved 97% MNIST test accuracy with 81% logic element utilisation using time-multiplexed MAC and fixed-point arithmetic.

Shape Classifier – FPGA + Python Pipeline

No Neural Network. No ML Library. Just Math, Hardware & One Very Stubborn Bug.

A real-time shape-classification system: a Python GUI streams a hand-drawn 28×28 bitmap over UART to an Altera DE2 FPGA, which classifies the shape in 20ns using purely combinational Verilog and fixed-point arithmetic. The FPGA extracts 5 geometric features (area, aspect ratio, row variance, symmetry, perimeter) via an FSM, then applies a nearest-centroid weighted L1 distance classifier — all in a single clock cycle. No floating-point units, no embedded multipliers, no DSP blocks. Supports Circle, Square, Rectangle, and Triangle.

RISC-V SoC on Vicharak Shrike FPGA

From Zero to Command Prompt with Spinning 3D Donut · $4 FPGA · 420 LUTs (37%)

Built a complete RISC-V computer on a $4 Vicharak Shrike FPGA board using the SERV (SErial RISC-V) CPU — the world's smallest RISC-V processor. The SoC includes a SERV CPU core (200 LUTs), 4KB RAM, UART (115200 baud), a timer, and LED control — all on a Wishbone bus — using just 420 of 1120 LUTs (37%).
